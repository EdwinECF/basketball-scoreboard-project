# Basketball Scoreboard

A simple and interactive basketball scoreboard built with HTML, CSS, and JavaScript. The application allows users to keep track of the home and guest team scores through an easy-to-use interface.

## Features

* Increase the score by 1, 2, or 3 points
* Separate scoreboards for the home and guest teams
* Instant score updates without refreshing the page
* Clean and responsive user interface
* Retro-inspired digital scoreboard design

## Built With

* HTML5
* CSS3
* JavaScript (ES6)
